== Description ==

The **Front-End Post Submission (FEPS)** plugin allows WordPress users to submit posts directly from the front-end of the website. It provides an easy-to-use form with the option to submit content using the block editor or the classic editor.

**Features:**
- Front-end post submission for users without access to the WordPress backend.
- Supports both the Block Editor (Gutenberg) and Classic Editor.
- Customizable post types and submission settings.
- Option to redirect users to a custom "Thank You" page after submission.

Whether you want to allow users to contribute to your site or collect blog posts from contributors, FEPS makes it easy for them to submit content directly from the front end.

== Installation ==

1. Upload the `feps-plugin` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. After activation, configure the plugin settings under the 'FEPS Settings' menu in the WordPress Admin area.

== Frequently Asked Questions ==

= How do I enable the block editor for front-end submission? =

You can enable the Block Editor for front-end submission by going to the FEPS Settings page in your WordPress dashboard. There, you can choose between the Classic Editor and the Block Editor.

= Can I restrict the posts submitted to specific users? =

Yes, the plugin can restrict the posts to the logged-in user only, ensuring that each user can only submit and view their own posts.

= Will the plugin work with custom post types? =

Yes, you can configure the plugin to submit posts to custom post types. This can be set in the FEPS Settings.

= Can I customize the "Thank You" page? =

Yes, after submitting a post, users can be redirected to a "Thank You" page. You can set this page from the FEPS Settings in the WordPress admin.
